UX (User Experience) Design

What is UX?

UX focuses on how users feel when using a website or application. It’s about ease of use, navigation flow, and user satisfaction. A good UX makes websites intuitive and enjoyable.

My Study Method for UX

For UX, I follow these steps: Watch UX tutorials → especially about user journeys and usability. Analyze real websites → checking what makes them easy or hard to use. 

Hands-on Practice → I redesign existing websites in
        Figma to improve user flow. Courses with Certification → I use NMSkills
        (LinkedIn Learning) and Coursera to get structured training.

Conclusion

Frontend development, UI, and UX are deeply connected: Frontend is the code. UI is the design.
